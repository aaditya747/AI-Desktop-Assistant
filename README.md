# AI Desktop Assistant for Deaf and Mute People

## Project Overview

An AI-powered desktop assistant developed using Python to improve accessibility for deaf and mute users through gesture-based interaction.

## Features

* Gesture recognition
* User-friendly desktop interface
* Accessibility-focused design
* Real-time communication support

## Technologies Used

* Python
* OpenCV
* MediaPipe
